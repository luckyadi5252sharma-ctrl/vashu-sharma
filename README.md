Indian Professional College Website is a fully responsive and modern web project built using HTML, CSS, and JavaScript. This website provides detailed information about the college including admissions, courses, facilities, faculty, placements, events, and gallery.

It features a clean professional UI, interactive facility popups, image gallery viewer, smooth navigation, and mobile-friendly design. The goal of this project is to create a real-world college website experience for students and visitors.

Technologies Used:
- HTML5
- CSS3
- JavaScript

Features:
- Responsive Design
- Interactive Sections
- Facilities Popup System
- Image Gallery Viewer
- Google Map Integration
